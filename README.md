# 🍰 ElyDoces - Site de Divulgação

## 📌 Sobre o Projeto

O **ElyDoces** é um site desenvolvido com o objetivo de divulgar produtos artesanais como bolos, pudins e dindins gourmet, além de facilitar o contato direto com clientes por meio do WhatsApp.

O projeto foi criado com foco em **simplicidade, usabilidade e design atrativo**, permitindo que pequenos negócios tenham presença online de forma eficiente.

---

## 🎯 Objetivo

Desenvolver uma página web estática que:

* Apresente os produtos da cliente
* Destaque itens principais (como tortas especiais)
* Facilite o contato via WhatsApp
* Divulgue redes sociais (Instagram)

---

## 🧠 Levantamento de Requisitos

O levantamento foi realizado por meio de uma **entrevista informal com a cliente**, onde foram identificadas as seguintes necessidades:

* Exibição de produtos com imagens e descrições
* Layout simples e organizado
* Botão de contato direto (WhatsApp)
* Divulgação do Instagram
* Design responsivo

---

## 🎨 Protótipo

Foi definido um protótipo básico contendo:

* Cabeçalho (nome da marca)
* Seção de apresentação
* Destaque de produto
* Grade de produtos
* Área de contato

Esse planejamento facilitou a organização do desenvolvimento.

---

## 💻 Tecnologias Utilizadas

* **HTML5** → Estrutura da página
* **CSS3** → Estilização e layout
* **JavaScript** → Interações (botão de contato)

---

## ⚙️ Funcionalidades

* 📱 Layout responsivo
* 🍩 Exibição de produtos em formato de grade
* 🍫 Destaque para produtos especiais
* 📞 Botão direto para WhatsApp
* 📸 Link para Instagram
